# Frontend

Next.js (App Router) dashboard for monthly summaries, reconciliation, and exports.

## Setup
```bash
cd frontend
pnpm install
pnpm dev
```

## Structure
- `app/` – routes
- `components/` – reusable UI widgets
- `lib/` – API clients
- `styles/` – Tailwind config

## Key pages
- `/dashboard`
- `/transactions`
- `/settings`
